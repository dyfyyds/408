# Git 提交指南

本文档说明如何使用中文提交代码。

## 提交模板

已配置中文提交模板，路径：`.gitmessage`

## 常用命令

- `git ci` - 提交更改（会打开编辑器）
- `git ca` - 修改最后一次提交（不修改提交信息）
- `git st` - 查看状态（简洁模式）
- `git lg` - 查看提交历史（简洁图形化）
- `git ll` - 查看提交历史（详细信息）

## 提交示例

```
feat(用户管理): 添加用户注册功能

- 实现用户注册表单验证
- 添加邮箱验证功能
- 集成第三方登录

Closes #123
```

## 提交类型

- **feat**: 新功能
- **fix**: 修复bug
- **docs**: 文档更新
- **style**: 代码格式修改
- **refactor**: 代码重构
- **perf**: 性能优化
- **test**: 测试相关
- **chore**: 构建/工具链修改

## 编辑器

默认使用 VS Code 打开提交信息编辑器。
