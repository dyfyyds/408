# 输入输出系统

> 本章核心是主机如何控制外设：I/O 接口、端口编址、程序查询、中断、DMA。

## 考点导航

| 考点 | 先看 | 做题抓手 |
|---|---|---|
| I/O 接口 | [[wiki/concepts/计算机组成原理/第7章 输入输出系统/IO接口\|IO接口]] | 数据缓冲、状态/控制寄存器、端口编址 |
| 程序查询 | [[wiki/concepts/计算机组成原理/第7章 输入输出系统/IO方式\|IO方式]] | CPU 主动轮询，效率低 |
| 中断方式 | [[wiki/concepts/计算机组成原理/第5章 中央处理器/异常与中断\|异常与中断]] | 请求、判优、响应、服务、返回 |
| DMA | [[wiki/concepts/计算机组成原理/第7章 输入输出系统/DMA方式\|DMA方式]] | 预处理、传送、后处理 |

## 本章要会

- 会比较程序查询、程序中断、DMA 对 CPU 参与程度的差异。
- 会区分统一编址和独立编址。
- 会说明 DMA 控制器的主要寄存器与传送阶段。

## 易错点

- DMA 传送数据块时主要由 DMA 控制器控制，总线仍可能与 CPU 竞争。
- 中断方式适合低速随机事件，DMA 适合高速成块数据传送。
