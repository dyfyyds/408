# 数据的表示和运算

## 数制与编码

### 进位计数制

#### 十进制
- 基数为10，使用0-9十个数字

#### 二进制
- 基数为2，使用0和1两个数字
- 计算机内部使用

#### 八进制
- 基数为8，使用0-7八个数字

#### 十六进制
- 基数为16，使用0-9和A-F十六个数字

### 进制转换

#### R进制转十进制
$$\sum_{i=-m}^{n-1} x_i \cdot R^i$$

#### 十进制转R进制
- 整数部分：除R取余，逆序排列
- 小数部分：乘R取整，顺序排列

## 机器数的表示

### 原码、反码、补码、移码

#### 定点数表示
定点数：小数点的位置固定

| 类型 | 定义 | 范围（n位，含1位符号） |
|------|------|----------------------|
| **原码** | 符号位+数值位 | $-(2^{n-1}-1) \sim +(2^{n-1}-1)$ |
| **反码** | 原码符号位不变，数值位取反 | 同原码 |
| **补码** | 反码+1 | $-2^{n-1} \sim +(2^{n-1}-1)$ |

#### 原码（Sign-Magnitude）
$$[x]_原 = \begin{cases}
x & x \ge 0, \text{ 符号位为0} \\
2^{n-1} - |x| & x < 0, \text{ 符号位为1}
\end{cases}$$

- 0的表示：+0 = 0000...0，-0 = 1000...0
- 存在两个0表示

#### 反码（Ones' Complement）
$$[x]_反 = \begin{cases}
x & x \ge 0 \\
(2^{n-1} - 1) - |x| & x < 0
\end{cases}$$

- 数值位按位取反
- 0的表示：+0 = 0000...0，-0 = 1111...1
- 存在两个0表示

#### 补码（Two's Complement）
$$[x]_补 = \begin{cases}
x & x \ge 0 \\
2^n - |x| & x < 0
\end{cases}$$

- 负数 = 模 - 绝对值
- 0的表示：唯一，0000...0
- **计算机内部普遍使用**

#### 移码（Excess/Offset）
$$[x]_移 = 2^{n-1} + x \quad (-2^{n-1} \le x \le 2^{n-1}-1)$$

- 主要用于表示浮点数的阶码
- 便于比较大小（直接看作无符号数）

#### 总结：四种码的特点

| 特性 | 原码 | 反码 | 补码 | 移码 |
|------|------|------|------|------|
| **0的表示** | 两个0 | 两个0 | 一个0 | 一个0 |
| **符号位处理** | 单独处理 | 单独处理 | 参与运算 | 参与运算 |
| **加减运算** | 需判断符号 | 需判断符号 | 统一处理 | 不适合加减 |
| **应用** | 基本不用 | 基本不用 | ✅ 计算机使用 | 浮点阶码 |

### 定点数运算

#### 定点加法
$$[x+y]_补 = [x]_补 + [y]_补 \pmod{2^n}$$

#### 定点减法
$$[x-y]_补 = [x]_补 + [-y]_补 = [x]_补 + [y]_补的相反数 \pmod{2^n}$$

#### 溢出判断

##### 方法一：双符号位法（变形补码）
- 双符号位：$s_1 s_2$
- 00：正数，11：负数
- 01：正溢出，10：负溢出
- 溢出条件：$s_1 \neq s_2$

##### 方法二：单符号位法
- 符号位的进位 $c_s$ 和最高位的进位 $c_1$ 异或
- 溢出条件：$c_s \oplus c_1 = 1$

### 定点乘法

#### 原码乘法（Booth算法）
- 符号位单独处理
- 数值位相乘，取绝对值

#### 补码乘法
- 符号位参与运算
- 采用Booth算法

## 浮点数表示

### 浮点数的格式
$$N = (-1)^S \times M \times R^E$$

- S：符号位
- M：尾数（有效数字）
- R：基数（通常为2）
- E：阶码（指数）

### IEEE 754标准

#### 单精度浮点数（32位）
```
┌──────┬──────┬──────────────────┐
│ S    │ Exp  │  Mantissa        │
│ 1位  │ 8位  │  23位            │
└──────┴──────┴──────────────────┘

偏置值 Bias = 127
真值 E = Exp - Bias = Exp - 127
```

#### 双精度浮点数（64位）
```
┌──────┬────────┬──────────────────┐
│ S    │ Exp    │  Mantissa        │
│ 1位  │ 11位   │  52位            │
└──────┴────────┴──────────────────┘

偏置值 Bias = 1023
真值 E = Exp - Bias = Exp - 1023
```

#### 规格化浮点数
- 尾数的最高位必须是1（隐藏位）
- 单精度：$1 \le M < 2$
- 双精度：$1 \le M < 2$

#### IEEE 754的特殊值

| 类型 | Exponent | 尾数 | 意义 |
|------|---------|------|------|
| **零** | 全0 | 全0 | ±0 |
| **非规格化数** | 全0 | 非全0 | 接近0的小数 |
| **规格化数** | 非全0且非全1 | 任意 | 普通浮点数 |
| **无穷大** | 全1 | 全0 | ±∞ |
| **NaN** | 全1 | 非全0 | 非数值（Not a Number） |

### 浮点数的精度

#### 十进制精度计算
- 单精度：约7位十进制有效数字（$2^{-23} \approx 10^{-7}$）
- 双精度：约15-16位十进制有效数字（$2^{-52} \approx 10^{-16}$）

#### 浮点数表示范围
- 单精度：约$10^{-38} \sim 10^{38}$
- 双精度：约$10^{-308} \sim 10^{308}$
