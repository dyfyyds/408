# 定点数与浮点数运算

## 定点数运算

### 定点加法运算

#### 补码加法规则
$$[x+y]_补 = [x]_补 + [y]_补 \pmod{2^n}$$

#### 加法步骤
1. 操作数用补码表示
2. 符号位参与运算
3. 两补码数相加，结果也是补码
4. 逢2进1（与二进制加法相同）

#### 补码加法示例
```
例：计算 15 + (-12) （假设用8位表示，其中1位符号位）

[15]_补 = 00001111
[-12]_补 = 11110100

  00001111
+ 11110100
  --------
1 00000011

取低8位：00000011，进位1丢弃

结果：[3]_补 = 00000011 ✓
```

### 定点减法运算

#### 补码减法规则
$$[x-y]_补 = [x]_补 + [-y]_补$$

即：减去一个数等于加上这个数的相反数

#### 减法步骤
1. 对减数[y]求相反数[-y]补
2. [x]补 + [-y]补
3. 结果即为[x-y]补

#### 补码减法示例
```
例：计算 15 - 12

[15]_补 = 00001111
[-12]_补 = 11110100

  00001111
+ 11110100
  --------
1 00000011

结果：[3]_补 = 00000011 ✓
```

### 溢出判断

#### 溢出的产生
当运算结果超出机器数所能表示的范围时，产生溢出

#### 溢出的条件
- 正数 + 正数 = 负数 → 正溢出
- 负数 + 负数 = 正数 → 负溢出

#### 溢出判断方法

##### 方法一：双符号位法（变形补码）
```
符号位：S1 S2

00 → 正数
11 → 负数
01 → 正溢出
10 → 负溢出

溢出条件：S1 ≠ S2
```

##### 方法二：单符号位法（进位判断）
```
符号位的进位 ⊕ 最高位的进位 = 1 → 溢出

溢出条件：Cs ⊕ C1 = 1
```

##### 方法三：判断符号位和进位
```
正数 + 正数 = 负数 → 溢出
负数 + 负数 = 正数 → 溢出
```

#### 溢出判断示例
```
例：计算 127 + 1 （用8位，1位符号，7位数值）

[127]_补 = 01111111
[1]_补 =  00000001

  01111111
+ 00000001
  --------
  10000000

符号位从0变为1，结果变成了负数

S1 = 0, S2 = 1，S1 ≠ S2 → 正溢出 ✗

正确结果应该是 128，但8位补码只能表示 -128 ~ +127
```

### 定点乘法运算

#### 原码乘法
- 符号位单独处理
- 数值位相乘，取绝对值

#### 补码乘法（Booth算法）
- 符号位参与运算
- 根据末位和附加位判断加减

### 定点除法运算

#### 补码除法
- 符号位单独处理
- 数值位相除

## 浮点数运算

### 浮点数的加减运算

#### 步骤

```
1. 对阶：使两个数的阶码相等
2. 尾数求和：对阶后，尾数相加（减）
3. 规格化：对结果进行规格化处理
4. 舍入：对规格化后的结果进行舍入
5. 溢出判断：检查是否溢出
```

#### 详细步骤

##### 第一步：对阶
$$ΔE = E_x - E_y$$

- 使小阶向大阶看齐
- 阶小的数的尾数右移，阶差个位置

```
例：x = 0.1101 × 2^01, y = -0.1011 × 2^11

E_x = 01, E_y = 11
ΔE = 01 - 11 = -10（即-2）

x的阶码小，x的尾数右移2位：

x → 0.001101 × 2^11
y → -0.1011 × 2^11
```

##### 第二步：尾数求和
```
  0.001101
+ -0.101100
  --------
  -0.011011
```

##### 第三步：规格化
- 规格化条件：尾数的最高位与符号位相反
- 规格化形式：0.1xxxxx 或 1.0xxxxx

```
-0.011011 不是规格化数

左移2位：
-0.1011 × 2^(11-2) = -0.1011 × 2^01

规格化完成 ✓
```

##### 第四步：舍入
保留指定位数时，需要对尾数进行舍入

###### 常用舍入方法

1. **截断法（Truncation）**
   - 直接截断多余位
   - 简单，但误差大

2. **就近舍入法（Round to Nearest）**
   - 四舍五入
   - IEEE 754标准采用

3. **朝零舍入（Round toward Zero）**
   - 向0靠近的方向舍入

4. **朝正无穷舍入（Round toward +∞）**
   - 向正无穷方向舍入

5. **朝负无穷舍入（Round toward -∞）**
   - 向负无穷方向舍入

##### 第五步：溢出判断
- 检查阶码是否溢出
- 阶码全1 → 上溢
- 阶码全0 → 下溢

### 浮点数的乘除运算

#### 乘法运算
$$z = x \times y$$

$$E_z = E_x + E_y$$

$$M_z = M_x \times M_y$$

```
例：x = 0.1101 × 2^01, y = 0.1011 × 2^11

E_z = 01 + 11 = 100（4位阶码，可能溢出）
M_z = 0.1101 × 0.1011 = 0.10001111

规格化、舍入、判断溢出
```

#### 除法运算
$$z = x / y$$

$$E_z = E_x - E_y$$

$$M_z = M_x / M_y$$

### 浮点数的精度

#### 十进制精度
- **单精度浮点数（32位）**：约7位十进制有效数字
- **双精度浮点数（64位）**：约15-16位十进制有效数字

#### 精度损失
浮点数不能精确表示所有实数，存在精度损失

```
例：0.1 在二进制中是无限循环小数

0.1 = 0.00011001100110011...（二进制）

计算机只能用有限位表示，存在精度误差
```
