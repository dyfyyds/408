# 存储器层次结构

## Definition

存储器层次结构用多级存储器在速度、容量、成本之间折中，典型顺序为寄存器、Cache、主存、辅存。

## How It Works

- 越靠近 CPU，速度越快、容量越小、单位成本越高。
- Cache 利用时间局部性和空间局部性。
- 虚拟存储器在主存和辅存之间建立逻辑扩展。

## Exam Focus

- 命中率、平均访问时间。
- Cache 与主存映射。
- 主存容量、地址位数和编址单位。

## Common Pitfalls

- Cache 透明于程序员，虚拟存储器对程序员呈现更大地址空间。
- 地址位数取决于可寻址单元数，不只取决于字长。
