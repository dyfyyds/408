# 计算机系统概述

## Definition

计算机系统由**硬件**和**软件**两大部分组成，采用**存储程序**的工作原理。

## 计算机的基本组成

### 冯诺依曼体系结构

#### 五大部件
1. **运算器（Arithmetic Logic Unit，ALU）**
   - 执行算术运算和逻辑运算

2. **控制器（Control Unit，CU）**
   - 控制计算机各部件协调工作
   - 从存储器取出指令，分析指令，发出控制信号

3. **存储器（Memory）**
   - 存放数据和程序
   - 分为主存和辅存

4. **输入设备（Input Device）**
   - 将外界信息输入计算机（如键盘、鼠标）

5. **输出设备（Output Device）**
   - 将计算机处理结果输出（如显示器、打印机）

#### 存储程序原理
- 程序和数据都存放在存储器中
- 计算机自动从存储器中取出指令并执行
- 这是现代计算机工作原理的基础

### 硬件和软件

#### 硬件（Hardware）
计算机系统中看得见、摸得着的物理实体

#### 软件（Software）
硬件之上的程序和数据

**软件分类：**
- **系统软件**：管理计算机资源（操作系统、编译器）
- **应用软件**：完成特定任务（Word、浏览器）

### 层次结构

```
应用软件
↓
系统软件（操作系统）
↓
机器语言（指令集）
↓
微程序/固件
↓
硬件（逻辑电路）
```

## 计算机的性能指标

### 主频
- CPU的时钟频率，单位为Hz
- 主频越高，CPU速度越快

### CPI（Cycles Per Instruction）
- 执行一条指令所需的时钟周期数
- CPI = 执行某段程序的时钟周期数 / 程序包含的指令条数

### IPC（Instructions Per Cycle）
- 每个时钟周期执行的指令条数
- IPC = 1 / CPI

### CPU执行时间
$$CPU执行时间 = \frac{指令条数 \times CPI}{主频}$$

### MIPS（Million Instructions Per Second）
每秒执行多少百万条指令
$$MIPS = \frac{指令条数}{执行时间 \times 10^6} = \frac{主频}{CPI \times 10^6}$$

### FLOPS（Floating-point Operations Per Second）
每秒执行多少次浮点运算

## 计算机的层次结构

### 微程序机器
- 由微程序解释执行机器指令
- 微指令 = 若干微命令的集合

### 传统机器
- 由微程序解释机器指令
- 机器指令 = 微指令的集合

### 虚拟机器
- 用操作系统提供的指令解释作业
- 操作系统虚拟机

### 高级语言虚拟机
- 用编译程序产生的机器语言程序解释执行
- 高级语言翻译成机器语言

## 考试要点

### 高频考点
1. **冯诺依曼体系结构** — 五大部件、存储程序原理
2. **计算机性能指标** — CPU执行时间、MIPS、CPI的计算
3. **计算机的层次结构** — 从微程序到高级语言的层次

### 常见陷阱
- CPI不是常数，不同指令的CPI不同
- MIPS与主频成正比，与CPI成反比
- FLOPS是衡量浮点运算性能的指标
