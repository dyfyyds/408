# 中断系统

## Definition

中断是 CPU 暂停当前程序，转去处理中断服务程序，处理完后再返回原程序的机制。

## How It Works

1. 中断请求。
2. 中断判优。
3. 中断响应，保存断点和现场。
4. 执行中断服务程序。
5. 恢复现场并返回。

## Exam Focus

- 中断响应条件和时机。
- 中断向量与入口地址。
- 多重中断与优先级。
- 与 DMA 的关系。

## Common Pitfalls

- 中断隐指令由硬件完成，不是指令系统中的一条普通指令。
- 保存断点和保存现场不是同一层次。

- 中断隐指令通常完成关中断、保存断点、引出中断服务程序入口。
- 多重中断要满足服务程序中开中断，并且新请求优先级更高。
