# DMA方式

## 定义

DMA 是直接存储器存取方式，由 DMA 控制器在外设和主存之间直接成块传送数据，CPU 主要参与初始化和结束处理。

## 机制/流程

1. 预处理：CPU 设置主存地址、传送长度、传送方向等。
2. 数据传送：DMA 控制器取得总线控制权，完成主存与外设之间的数据传送。
3. 后处理：传送结束后 DMA 发出中断，请 CPU 做收尾处理。

## 核心判断表

| 传送方式 | 特点 |
|---|---|
| 停止 CPU 访存 | DMA 传送时 CPU 暂停访存 |
| 周期挪用 | DMA 挪用一个或若干主存周期 |
| 交替访存 | CPU 和 DMA 分时交替访问主存 |

## 考点题型

- DMA 控制器组成部件判断。
- DMA 与中断方式比较。
- DMA 传送流程排序。

## 易错点

- DMA 不是完全不需要 CPU，初始化和结束处理仍需 CPU。
- DMA 和 CPU 共享主存和总线，可能发生总线竞争。
