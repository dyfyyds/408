# 拓扑排序

## Definition

拓扑排序是对有向无环图的顶点排序，使每条有向边的起点都排在终点之前。

## How It Works

1. 选择入度为 0 的顶点输出。
2. 删除该顶点及其出边。
3. 重复直到所有顶点输出或不存在入度为 0 的顶点。

## Exam Focus

- 判断图是否有环。
- 手工给出一种或全部拓扑序列。
- 与关键路径中的事件最早/最迟时间计算衔接。

## Common Pitfalls

- 拓扑序列可能不唯一。
- 只有 DAG 才存在拓扑排序。
