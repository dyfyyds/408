# 栈和队列（Stack & Queue）

## Definition

**栈（Stack）**：只允许在一端进行插入或删除的线性表，遵循**后进先出（LIFO）**原则

**队列（Queue）**：只允许在一端插入（队尾），另一端删除（队头）的线性表，遵循**先进先出（FIFO）**原则

## 栈（Stack）

### 基本概念
- **栈顶**：允许插入和删除的一端
- **栈底**：不允许插入和删除的一端
- **空栈**：不含任何元素的栈

### 顺序栈
```c
#define MaxSize 100
typedef struct {
    ElemType data[MaxSize];
    int top;    // 栈顶指针
} SqStack;
```

#### 入栈（Push）
```c
bool Push(SqStack &S, ElemType x) {
    if (S.top == MaxSize - 1)
        return false;  // 栈满
    S.data[++S.top] = x;
    return true;
}
```

#### 出栈（Pop）
```c
bool Pop(SqStack &S, ElemType &x) {
    if (S.top == -1)
        return false;  // 栈空
    x = S.data[S.top--];
    return true;
}
```

#### 共享栈
- 两个栈共享一个数组空间
- 两个栈的栈顶指针相向增长
- 充分利用空间，防止空间浪费

### 链栈
```c
typedef struct LinkNode {
    ElemType data;
    struct LinkNode *next;
} LinkNode, *LinkStack;
```

### 栈的应用
1. **括号匹配** — 表达式求值
2. **表达式求值** — 中缀转后缀，利用栈求值
3. **递归实现** — 函数调用栈
4. **深度优先遍历** — 图的DFS

## 队列（Queue）

### 基本概念
- **队头**：允许删除的一端
- **队尾**：允许插入的一端
- **空队列**：不含任何元素的队列

### 顺序队列
```c
typedef struct {
    ElemType data[MaxSize];
    int front, rear;  // 队头和队尾指针
} SqQueue;
```

### 循环队列
```c
// 初始化
void InitQueue(SqQueue &Q) {
    Q.front = Q.rear = 0;
}

// 入队
bool EnQueue(SqQueue &Q, ElemType x) {
    if ((Q.rear + 1) % MaxSize == Q.front)
        return false;  // 队满
    Q.data[Q.rear] = x;
    Q.rear = (Q.rear + 1) % MaxSize;
    return true;
}

// 出队
bool DeQueue(SqQueue &Q, ElemType &x) {
    if (Q.rear == Q.front)
        return false;  // 队空
    x = Q.data[Q.front];
    Q.front = (Q.front + 1) % MaxSize;
    return true;
}

// 队空判断
bool QueueEmpty(SqQueue Q) {
    return Q.front == Q.rear;
}

// 队满判断（牺牲一个存储单元）
bool QueueFull(SqQueue Q) {
    return (Q.rear + 1) % MaxSize == Q.front;
}

// 队列长度
int QueueLength(SqQueue Q) {
    return (Q.rear - Q.front + MaxSize) % MaxSize;
}
```

#### 循环队列的特点
- 牺牲一个存储单元来判断队满
- 队空条件：`front == rear`
- 队满条件：`(rear + 1) % MaxSize == front`
- 队列长度：`(rear - front + MaxSize) % MaxSize`

### 链队列
```c
typedef struct {
    LinkNode *front, *rear;  // 队头和队尾指针
} LinkQueue;
```

### 双端队列（Deque）
- 允许两端插入和删除的队列
- **输入受限的双端队列**：允许删除两端，插入只允许在一端
- **输出受限的双端队列**：允许插入两端，删除只允许在一端

## 特殊队列

### 优先级队列
- 出队顺序由优先级决定
- 可以用堆（Heap）实现

### 双端队列（Double-ended Queue）
- 可以在两端进行插入和删除操作

## 栈和队列的对比

| 对比维度 | 栈 | 队列 |
|---------|-----|------|
| **操作规则** | 后进先出（LIFO） | 先进先出（FIFO） |
| **入队/入栈** | 栈顶 | 队尾 |
| **出队/出栈** | 栈顶 | 队头 |
| **主要应用** | 递归、表达式求值 | 缓冲区、消息队列 |

## 考试要点

### 高频考点
1. **栈的基本操作** — 入栈出栈顺序
2. **队列的基本操作** — 循环队列的队满队空判断
3. **循环队列的元素个数** — `(rear - front + MaxSize) % MaxSize`
4. **栈的应用场景** — 括号匹配、递归

### 常见陷阱
- 循环队列不能通过`rear == front`判断队满
- 队列元素个数的计算（注意取模）

- 2014年408真题：循环队列元素个数计算
