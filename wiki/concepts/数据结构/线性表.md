# 线性表（Linear List）

## Definition

**线性表**是n个具有相同特性的数据元素的有限序列。

线性表是一种逻辑结构（逻辑特性），可以有两种存储结构：
1. **顺序存储结构** → 顺序表（物理上相邻）
2. **链式存储结构** → 链表（物理上不一定相邻）

## 线性表的基本操作

### 初始化
```c
Status InitList(SqList &L);
```

### 基本操作
```c
// 插入：在位置i插入元素e
Status ListInsert(SqList &L, int i, ElemType e);

// 删除：删除位置i的元素，用e返回
Status ListDelete(SqList &L, int i, ElemType &e);

// 按位查找：返回第i个元素的值
ElemType GetElem(SqList L, int i);

// 按值查找：查找值为e的元素，返回位序
int LocateElem(SqList L, ElemType e);
```

## 线性表的顺序存储

### 顺序表（SqList）
```c
#define MaxSize 100
typedef struct {
    ElemType data[MaxSize];  // 用静态数组存放数据元素
    int length;              // 当前长度
} SqList;
```

### 顺序表的特点
- **随机访问**：通过下标可以直接访问元素，时间复杂度O(1)
- **存储密度高**：每个节点只存储数据元素
- **插入删除不便**：需要移动大量元素，时间复杂度O(n)

### 顺序表的插入
```c
// 最好情况：在表尾插入，时间复杂度 O(1)
// 最坏情况：在表头插入，时间复杂度 O(n)
// 平均情况：时间复杂度 O(n)
Status ListInsert(SqList &L, int i, ElemType e) {
    if (i < 1 || i > L.length + 1)
        return ERROR;
    if (L.length >= MaxSize)
        return ERROR;
    
    for (int j = L.length; j >= i; j--)
        L.data[j] = L.data[j - 1];  // 从后往前移动
    
    L.data[i - 1] = e;
    L.length++;
    return OK;
}
```

### 顺序表的删除
```c
// 时间复杂度：O(n)
Status ListDelete(SqList &L, int i, ElemType &e) {
    if (i < 1 || i > L.length)
        return ERROR;
    
    e = L.data[i - 1];
    
    for (int j = i; j < L.length; j++)
        L.data[j] = L.data[j + 1];  // 从i开始，往前移动
    
    L.length--;
    return OK;
}
```

## 线性表的链式存储

### 单链表（LinkList）
```c
typedef struct LNode {        // 定义单链表节点类型
    ElemType data;            // 数据域
    struct LNode *next;       // 指针域
} LNode, *LinkList;
```

### 单链表的特点
- **非随机访问**：必须从头节点开始遍历，时间复杂度O(n)
- **存储密度低**：每个节点除了数据元素还要存储指针
- **插入删除方便**：不需要移动元素，时间复杂度O(1)（已知位置）

### 单链表的头插法与尾插法

#### 头插法
```c
// 时间复杂度：O(n)
LinkList List_HeadInsert(LinkList &L) {
    L = (LinkList)malloc(sizeof(LNode));
    L->next = NULL;
    
    LNode *s;
    int x;
    scanf("%d", &x);
    while (x != 9999) {
        s = (LNode*)malloc(sizeof(LNode));
        s->data = x;
        s->next = L->next;
        L->next = s;
        scanf("%d", &x);
    }
    return L;
}
```

#### 尾插法（更常用）
```c
// 时间复杂度：O(n)
LinkList List_TailInsert(LinkList &L) {
    L = (LinkList)malloc(sizeof(LNode));
    L->next = NULL;
    
    LNode *s, *r = L;  // r指向尾节点
    int x;
    scanf("%d", &x);
    while (x != 9999) {
        s = (LNode*)malloc(sizeof(LNode));
        s->data = x;
        r->next = s;
        r = s;
        scanf("%d", &x);
    }
    r->next = NULL;
    return L;
}
```

### 单链表的插入
```c
// 时间复杂度：O(1)
// 在p节点之后插入节点s
s->next = p->next;
p->next = s;
```

### 单链表的删除
```c
// 时间复杂度：O(1)
// 删除p节点的后继节点
p->next = p->next->next;
free(p->next);  // 释放节点
```

## 双链表

### 结构定义
```c
typedef struct DulNode {
    ElemType data;
    struct DulNode *prior;  // 前驱指针
    struct DulNode *next;   // 后继指针
} DulNode, *DuLinkList;
```

### 双链表的特点
- 每个节点有两个指针，分别指向前驱和后继
- 可以双向遍历
- 插入删除更加灵活

## 循环链表

### 特点
- 尾节点的指针指向头节点
- 从任意节点出发可以访问到表中所有节点
- 不需要判断是否为最后一个节点

## 静态链表

### 特点
- 用数组来实现链表
- 通过数组下标来模拟指针
- 适合于不支持指针的语言（如Basic）

## 顺序表 vs 链表

| 对比维度 | 顺序表 | 单链表 |
|---------|--------|--------|
| **存储空间** | 必须连续 | 可以不连续 |
| **随机访问** | ✅ O(1) | ❌ O(n) |
| **插入删除** | ❌ O(n) | ✅ O(1)（已知位置） |
| **存储密度** | 高（1） | 低（<1） |
| **容量变化** | 固定（静态）或动态数组 | 灵活，可按需分配 |
| **操作复杂度** | 简单 | 稍复杂 |
