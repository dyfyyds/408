# 磁盘调度

> 磁盘存储结构、访问时间计算及磁盘调度算法。详见 [[虚拟内存与磁盘调度]]。

---

## 磁盘的结构

### 盘片
- 磁盘由多个盘片组成
- 每个盘片有两个盘面

### 磁道
- 盘面上的同心圆，编号 0, 1, 2, ...

### 扇区
- 磁道划分为多个扇区
- 每个扇区 512 字节或 4KB

### 柱面
- 所有盘面上的同一磁道组成的圆柱面

## 磁盘访问时间

$$磁盘访问时间 = 寻道时间 + 旋转延迟时间 + 传输时间$$

| 组成部分 | 说明 |
|----------|------|
| **寻道时间** | 磁头移动到目标磁道的时间 |
| **旋转延迟** | 等待目标扇区到达磁头下的时间，平均 = 半圈时间 = (60/转速)/2 |
| **传输时间** | 数据从一个扇区传送到内存的时间 |

## 磁盘调度算法

### 1. 先来先服务（FCFS）
- 按请求到达顺序处理
- 公平但性能较差

### 2. 最短寻道时间优先（SSTF）
- 优先处理距当前磁头最近的请求
- 可能出现**饥饿**现象

### 3. 扫描算法（SCAN，电梯算法）
- 磁头从一端移动到另一端，沿途响应请求
- 到达一端后反向移动

### 4. 循环扫描算法（C-SCAN）
- 磁头只沿一个方向扫描
- 到达一端后直接返回起点

### 5. LOOK 和 C-LOOK
- SCAN 和 C-SCAN 的改进
- 磁头移动到最远请求处就反向
