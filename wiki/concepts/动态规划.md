# 动态规划（Dynamic Programming）

## 什么是动态规划？

动态规划（DP）是一种**分治**思想，通过把原问题分解为若干个子问题，求解子问题并存储子问题的解，避免重复计算，从而求解原问题。

### 核心思想

#### 1. 最优子结构
原问题的最优解包含子问题的最优解

#### 2. 重叠子问题
在求解过程中，很多子问题是重复的

#### 3. 无后效性
一旦某个状态确定后，之后的过程只与当前状态有关，与到达该状态的路径无关

### 动态规划的步骤

```
1. 定义状态：dp[i]表示什么含义？
2. 状态转移方程：dp[i]如何由其他状态推导而来？
3. 边界条件：初始状态是什么？
4. 计算顺序：按照什么顺序计算？
5. 返回结果：最终结果是什么？
```

## 背包问题

### 0-1背包问题

#### 问题描述
有N件物品和一个容量为V的背包。第i件物品的重量是w[i]，价值是v[i]。求解将哪些物品装入背包可使这些物品的重量总和不超过背包容量，且价值总和最大。

#### 状态定义
$$dp[i][j]$$ 表示从前i件物品中选择，放入容量为j的背包中可以获得的最大价值

#### 状态转移方程
```
不放第i件物品：dp[i][j] = dp[i-1][j]
放第i件物品：   dp[i][j] = dp[i-1][j-w[i]] + v[i]

dp[i][j] = max(dp[i-1][j], dp[i-1][j-w[i]] + v[i])
```

#### 代码实现
```c
int knapsack_01(int N, int V, int w[], int v[]) {
    // dp[i][j]表示前i件物品放入容量为j的背包的最大价值
    int dp[N+1][V+1];
    
    // 初始化
    for (int i = 0; i <= N; i++)
        dp[i][0] = 0;
    for (int j = 0; j <= V; j++)
        dp[0][j] = 0;
    
    // 动态规划
    for (int i = 1; i <= N; i++) {
        for (int j = 1; j <= V; j++) {
            if (j < w[i]) {
                dp[i][j] = dp[i-1][j]; // 放不下
            } else {
                dp[i][j] = max(dp[i-1][j], dp[i-1][j-w[i]] + v[i]);
            }
        }
    }
    
    return dp[N][V];
}
```

#### 空间优化
由于dp[i][j]只依赖于dp[i-1][...]，可以优化为一维数组

```c
int knapsack_01_optimized(int N, int V, int w[], int v[]) {
    int dp[V+1] = {0};
    
    for (int i = 1; i <= N; i++) {
        // 必须从后往前遍历，避免重复使用物品
        for (int j = V; j >= w[i]; j--) {
            dp[j] = max(dp[j], dp[j-w[i]] + v[i]);
        }
    }
    
    return dp[V];
}
```

### 完全背包问题

#### 问题描述
与0-1背包类似，但每种物品有**无限件**

#### 状态转移方程
```c
// 完全背包：可以重复选择第i件物品
for (int j = w[i]; j <= V; j++) {
    dp[j] = max(dp[j], dp[j-w[i]] + v[i]);
}
```

#### 0-1背包 vs 完全背包

| 对比 | 0-1背包 | 完全背包 |
|------|---------|---------|
| **物品数量** | 每件最多1件 | 每件无限件 |
| **内层循环方向** | 从大到小 | 从小到大 |
| **时间复杂度** | O(NV) | O(NV) |

### 多重背包问题

#### 问题描述
每种物品有**指定数量**num[i]

#### 转化为0-1背包
把num[i]件物品拆分成多件0-1背包的物品

```c
// 二进制拆分优化
for (int k = 1; num[i] > 0; k <<= 1) {
    int take = min(k, num[i]);
    // 将take件物品作为一件新的物品
    num[i] -= take;
}
```

## 最长公共子序列（LCS）

### 问题描述
给定两个字符串S1和S2，求它们的最长公共子序列的长度

### 状态定义
$$dp[i][j]$$ 表示S1的前i个字符与S2的前j个字符的最长公共子序列的长度

### 状态转移方程
```
如果S1[i-1] == S2[j-1]:
    dp[i][j] = dp[i-1][j-1] + 1
否则:
    dp[i][j] = max(dp[i-1][j], dp[i][j-1])
```

### 代码实现
```c
int LCS(char *S1, char *S2, int m, int n) {
    int dp[m+1][n+1];
    
    // 初始化
    for (int i = 0; i <= m; i++)
        dp[i][0] = 0;
    for (int j = 0; j <= n; j++)
        dp[0][j] = 0;
    
    // 动态规划
    for (int i = 1; i <= m; i++) {
        for (int j = 1; j <= n; j++) {
            if (S1[i-1] == S2[j-1]) {
                dp[i][j] = dp[i-1][j-1] + 1;
            } else {
                dp[i][j] = max(dp[i-1][j], dp[i][j-1]);
            }
        }
    }
    
    return dp[m][n];
}
```

### 时间复杂度和空间复杂度
- **时间复杂度**：O(m×n)
- **空间复杂度**：O(m×n)，可以优化到O(min(m, n))

## 最长递增子序列（LIS）

### 问题描述
给定一个序列，找到其中最长的严格递增子序列

### 方法一：动态规划

#### 状态定义
$$dp[i]$$ 表示以第i个元素结尾的最长递增子序列的长度

#### 状态转移方程
```
dp[i] = max(dp[j] + 1)  （对于所有j < i且a[j] < a[i]）
```

#### 代码实现
```c
int LIS(int a[], int n) {
    int dp[n];
    
    // 初始化
    for (int i = 0; i < n; i++)
        dp[i] = 1;
    
    // 动态规划
    for (int i = 1; i < n; i++) {
        for (int j = 0; j < i; j++) {
            if (a[j] < a[i]) {
                dp[i] = max(dp[i], dp[j] + 1);
            }
        }
    }
    
    // 返回最大值
    int maxLen = 0;
    for (int i = 0; i < n; i++)
        maxLen = max(maxLen, dp[i]);
    
    return maxLen;
}
```

#### 时间复杂度
- **时间复杂度**：O(n²)
- **空间复杂度**：O(n)

### 方法二：贪心 + 二分查找（O(nlogn)）

```c
int LIS_optimized(int a[], int n) {
    int dp[n]; // dp[i]表示长度为i+1的递增子序列的最小末尾元素
    
    int len = 0;
    for (int i = 0; i < n; i++) {
        // 二分查找第一个大于等于a[i]的位置
        int pos = lower_bound(dp, dp + len, a[i]);
        dp[pos] = a[i];
        
        if (pos == len) len++;
    }
    
    return len;
}
```

## 最长公共子串

### 问题描述
给定两个字符串S1和S2，求它们的最长公共子串（连续）的长度

### 状态定义
$$dp[i][j]$$ 表示以S1[i-1]和S2[j-1]结尾的最长公共子串的长度

### 状态转移方程
```
如果S1[i-1] == S2[j-1]:
    dp[i][j] = dp[i-1][j-1] + 1
否则:
    dp[i][j] = 0
```

### 代码实现
```c
int LCSubstring(char *S1, char *S2, int m, int n) {
    int dp[m+1][n+1];
    int maxLen = 0;
    
    for (int i = 0; i <= m; i++)
        dp[i][0] = 0;
    for (int j = 0; j <= n; j++)
        dp[0][j] = 0;
    
    for (int i = 1; i <= m; i++) {
        for (int j = 1; j <= n; j++) {
            if (S1[i-1] == S2[j-1]) {
                dp[i][j] = dp[i-1][j-1] + 1;
                maxLen = max(maxLen, dp[i][j]);
            } else {
                dp[i][j] = 0;
            }
        }
    }
    
    return maxLen;
}
```

### 2. 最大子段和
```c
int maxSubArray(int a[], int n) {
    int dp[n];
    dp[0] = a[0];
    int maxSum = a[0];
    
    for (int i = 1; i < n; i++) {
        dp[i] = max(a[i], dp[i-1] + a[i]);
        maxSum = max(maxSum, dp[i]);
    }
    
    return maxSum;
}
```

### 3. 编辑距离
求将一个字符串转换成另一个字符串所需的最少操作次数（插入、删除、替换）

```c
int editDistance(char *S1, char *S2, int m, int n) {
    int dp[m+1][n+1];
    
    for (int i = 0; i <= m; i++)
        dp[i][0] = i;
    for (int j = 0; j <= n; j++)
        dp[0][j] = j;
    
    for (int i = 1; i <= m; i++) {
        for (int j = 1; j <= n; j++) {
            if (S1[i-1] == S2[j-1]) {
                dp[i][j] = dp[i-1][j-1];
            } else {
                dp[i][j] = min(
                    dp[i-1][j-1] + 1, // 替换
                    min(dp[i-1][j] + 1,   // 删除
                        dp[i][j-1] + 1)  // 插入
                );
            }
        }
    }
    
    return dp[m][n];
}
```

## 考试要点

### 高频考点
1. **背包问题** — 0-1背包、完全背包、多重背包
2. **LCS问题** — 最长公共子序列
3. **LIS问题** — 最长递增子序列
4. **DP的核心思想** — 最优子结构、重叠子问题

### 常见陷阱
- 0-1背包内层循环从大到小
- 完全背包内层循环从小到大
- LCS与最长公共子串的区别
