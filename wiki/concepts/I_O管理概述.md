# I/O 管理概述

> I/O 设备分类与 I/O 控制方式。详见 [[文件系统与IO管理]]。

---

## I/O 设备分类

### 按传输速率分类
- **低速设备**：键盘、鼠标（几到几百 B/s）
- **中速设备**：打印机（几千到几万 B/s）
- **高速设备**：磁盘、网卡（几十万到几 MB/s）

### 按信息交换单位分类
- **块设备**：磁盘、SD 卡（以块为单位传输）
- **字符设备**：键盘、鼠标、显示器（以字符为单位传输）

### 按设备共享属性分类
- **独占设备**：打印机（一段时间内只能由一个进程独占）
- **共享设备**：磁盘（可被多个进程同时访问）
- **虚拟设备**：通过 SPOOLing 技术实现的虚拟设备

## I/O 控制方式

### 1. 程序查询方式
- CPU 主动查询 I/O 设备状态
- 串行工作，CPU 利用率低

### 2. 程序中断方式
- I/O 设备就绪时向 CPU 发中断
- CPU 与 I/O 并行工作

### 3. DMA 方式（Direct Memory Access）
- I/O 设备直接与内存交换数据
- CPU 仅在初始化和结束时介入

### 4. 通道控制方式
- 引入专门的 I/O 处理机（通道）
- CPU 只需发 I/O 指令，通道完成数据交换
- 适合大规模 I/O 系统
