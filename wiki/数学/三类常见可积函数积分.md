# 三类常见可积函数积分

在高数中，有三类特殊结构的函数积分是有固定求解套路的。熟练掌握这些套路不仅能解决直接考查的题目，也是处理复杂积分的必备基本功。

---

## 1. 有理函数积分 $\int R(x)\,dx$

两个多项式的商 $\frac{P(x)}{Q(x)}$ 称为**有理函数**。

### 核心方法：部分分式法（一般方法）
如果 $\frac{P(x)}{Q(x)}$ 是假分式（分子次数 $\ge$ 分母次数），应先通过**多项式除法**化为 `多项式 + 真分式` 的形式。
以下假设 $\frac{P(x)}{Q(x)}$ 为真分式。其积分步骤为：
1. **分母因式分解**：将 $Q(x)$ 完全分解为一次因式和二次不可约因式的乘积：
   $$Q(x) = (x-a)^n \cdots (x^2+px+q)^m$$
2. **部分分式展开**：写出对应的部分分式待定形式：
   - 对应一次因式 $(x-a)^n$，展开为：
     $$\frac{A_1}{x-a} + \frac{A_2}{(x-a)^2} + \cdots + \frac{A_n}{(x-a)^n}$$
   - 对应二次不可约因式 $(x^2+px+q)^m$（其中 $p^2-4q < 0$），展开为：
     $$\frac{B_1 x + C_1}{x^2+px+q} + \frac{B_2 x + C_2}{(x^2+px+q)^2} + \cdots + \frac{B_m x + C_m}{(x^2+px+q)^m}$$
3. **确定待定系数**：利用待定系数法（取特殊值法或比较系数法）解出所有的 $A_i, B_i, C_i$。
4. **分项积分**：有理真分式积分最终转化为以下四种基本形式的积分：
   - $\int \frac{A}{x-a}\,dx = A\ln|x-a| + C$
   - $\int \frac{A}{(x-a)^n}\,dx = -\frac{A}{(n-1)(x-a)^{n-1}} + C \quad (n \ge 2)$
   - $\int \frac{Ax+B}{x^2+px+q}\,dx$：通过分子**凑微分**（凑分母的导数 $2x+p$）和分母**配方**转化为对数和反正切函数。
   - $\int \frac{Ax+B}{(x^2+px+q)^n}\,dx$：（一般不要求高阶二次因式，可用递推公式或三角代换）。

### 特殊技巧
当分母因式分解较简单时，优先采用**加项减项法**或**凑微分降幂法**来快速分拆，避免繁琐的待定系数法。

---

### 有理函数典型例题

#### 【例 10】计算 $\int \frac{x+5}{x^2 - 6x + 13}\,dx$
**解析**：
分母无法进行一次实数因式分解（$\Delta = (-6)^2 - 4 \times 13 = -16 < 0$），属于不可约二次式。
1. **分母配方**：
   $$x^2 - 6x + 13 = (x-3)^2 + 4$$
2. **分子凑微分**：分母的导数为 $2x - 6$，我们需要在分子中凑出此项：
   $$x+5 = \frac{1}{2}(2x - 6) + 8$$
3. **拆分积分**：
   $$
   \begin{aligned}
   \int \frac{x+5}{x^2 - 6x + 13}\,dx &= \int \frac{\frac{1}{2}(2x-6) + 8}{x^2-6x+13}\,dx \\
   &= \frac{1}{2}\int \frac{2x-6}{x^2-6x+13}\,dx + 8\int \frac{dx}{(x-3)^2 + 2^2} \\
   &= \frac{1}{2}\ln(x^2 - 6x + 13) + 4\arctan\frac{x-3}{2} + C
   \end{aligned}
   $$

#### 【例 11】求不定积分 $\int \frac{x\,dx}{x^4 + 2x^2 + 5}$
**解析**：
分子有 $x\,dx$，让人联想到对 $x^2$ 进行凑微分。
1. **凑微分**：令 $u = x^2$，则 $du = 2x\,dx$。
2. **代入化简并配方**：
   $$
   \int \frac{x\,dx}{x^4 + 2x^2 + 5} = \frac{1}{2}\int \frac{d(x^2)}{(x^2)^2 + 2(x^2) + 5} = \frac{1}{2}\int \frac{du}{u^2 + 2u + 5}
   $$
   对分母配方得 $u^2 + 2u + 5 = (u+1)^2 + 4$。
3. **求积分**：
   $$
   \frac{1}{2}\int \frac{d(u+1)}{(u+1)^2 + 2^2} = \frac{1}{2} \cdot \frac{1}{2}\arctan\frac{u+1}{2} + C = \frac{1}{4}\arctan\frac{x^2+1}{2} + C
   $$

#### 【例 12】求不定积分 $\int \frac{3x+6}{(x-1)^2(x^2+x+1)}\,dx$
**解析**：
分母已因式分解，包含二重一次因式 $(x-1)^2$ 和不可约二次式 $x^2+x+1$。
1. **设部分分式待定形式**：
   $$\frac{3x+6}{(x-1)^2(x^2+x+1)} = \frac{A}{x-1} + \frac{B}{(x-1)^2} + \frac{Dx+E}{x^2+x+1}$$
2. **通分去分母**：
   $$3x+6 = A(x-1)(x^2+x+1) + B(x^2+x+1) + (Dx+E)(x-1)^2$$
3. **求解待定系数**：
   - 令 $x = 1$，代入得：$9 = 3B \implies B = 3$。
   - 比较 $x^3$ 系数：$A + D = 0 \implies D = -A$。
   - 令 $x = 0$，代入得：$6 = -A + B + E \implies 6 = -A + 3 + E \implies E - A = 3$。
   - 比较 $x^2$ 系数：由于 $A(x-1)(x^2+x+1) = A(x^3-1)$ 没有 $x^2$ 项，因此 $B + E - 2D = 0 \implies 3 + E + 2A = 0$。
   - 联立方程：
     $$
     \begin{cases}
     E - A = 3 \\
     E + 2A = -3
     \end{cases}
     \implies 3A = -6 \implies A = -2,\ E = 1
     $$
     从而得到 $D = 2$。
4. **分项积分**：
   $$
   \begin{aligned}
   \text{原式} &= \int \frac{-2}{x-1}\,dx + \int \frac{3}{(x-1)^2}\,dx + \int \frac{2x+1}{x^2+x+1}\,dx \\
   &= -2\ln|x-1| - \frac{3}{x-1} + \ln(x^2+x+1) + C
   \end{aligned}
   $$

## 2. 三角有理式积分 $\int R(\sin x, \cos x)\,dx$

由 $\sin x$ 和 $\cos x$ 的有理运算构成的积分。

### 一般方法：万能代换
令 $t = \tan\frac{x}{2}$（其中 $-\pi < x < \pi$），则有：
$$\sin x = \frac{2t}{1+t^2},\qquad \cos x = \frac{1-t^2}{1+t^2},\qquad dx = \frac{2}{1+t^2}\,dt$$
代入后，可将三角有理式积分完全转化为**有理函数积分**。
> **注意**：万能代换具有通用性，但往往计算量极大，应优先寻找特殊方法。

### 特殊方法：结构换元法
通过判断被积函数 $R(\sin x, \cos x)$ 的奇偶性选择更简便的换元：
1. **关于 $\sin x$ 呈奇对称**，即 $R(-\sin x, \cos x) = -R(\sin x, \cos x)$：
   - **方法**：令 $u = \cos x$（凑出 $d\cos x = -\sin x\,dx$）。
2. **关于 $\cos x$ 呈奇对称**，即 $R(\sin x, -\cos x) = -R(\sin x, \cos x)$：
   - **方法**：令 $u = \sin x$（凑出 $d\sin x = \cos x\,dx$）。
3. **关于 $\sin x, \cos x$ 呈同号偶对称**，即 $R(-\sin x, - \cos x) = R(\sin x, \cos x)$：
   - **方法**：令 $u = \tan x$（凑出 $d\tan x = \sec^2 x\,dx$）。

---

### 三角有理式典型例题

#### 【例 13】求 $\int \frac{dx}{1+\sin x}$
**解析**：
* **方法一（三角变形）**：
  分子分母同乘 $1 - \sin x$：
  $$
  \begin{aligned}
  \int \frac{1-\sin x}{1-\sin^2 x}\,dx &= \int \frac{1-\sin x}{\cos^2 x}\,dx \\
  &= \int (\sec^2 x - \sec x\tan x)\,dx \\
  &= \tan x - \sec x + C
  \end{aligned}
  $$
* **方法二（万能代换）**：
  令 $t = \tan\frac{x}{2}$，代入公式：
  $$
  \int \frac{1}{1+\frac{2t}{1+t^2}} \cdot \frac{2}{1+t^2}\,dt = \int \frac{2}{t^2+2t+1}\,dt = \int \frac{2\,dt}{(t+1)^2} = -\frac{2}{1+t} + C = -\frac{2}{1+\tan\frac{x}{2}} + C
  $$
  > **注**：两种方法形式不同但完全等价。因为：
  > $$\tan x - \sec x = \frac{\sin x - 1}{\cos x} = \frac{-(\cos\frac{x}{2}-\sin\frac{x}{2})^2}{\cos^2\frac{x}{2}-\sin^2\frac{x}{2}} = \frac{\sin\frac{x}{2}-\cos\frac{x}{2}}{\cos\frac{x}{2}+\sin\frac{x}{2}} = \frac{\tan\frac{x}{2}-1}{\tan\frac{x}{2}+1} = 1 - \frac{2}{1+\tan\frac{x}{2}}$$
  > 两者仅相差常数 $1$，已吸收到积分常数 $C$ 中。

#### 【例 14】求 $\int \frac{dx}{\sin(2x) + 2\sin x}$
**解析**：
被积函数关于 $\sin x$ 和 $\cos x$ 均为奇函数。
1. **化简结构**：
   $$\sin 2x + 2\sin x = 2\sin x\cos x + 2\sin x = 2\sin x(\cos x + 1)$$
2. **分子分母同乘 $\sin x$**：
   $$\int \frac{dx}{2\sin x(\cos x+1)} = \int \frac{\sin x\,dx}{2\sin^2 x(\cos x+1)} = \int \frac{\sin x\,dx}{2(1-\cos^2 x)(\cos x+1)}$$
3. **换元法**：令 $u = \cos x$，则 $du = -\sin x\,dx$。
   $$
   \text{原式} = -\frac{1}{2}\int \frac{du}{(1-u^2)(1+u)} = -\frac{1}{2}\int \frac{du}{(1-u)(1+u)^2}
   $$
4. **部分分式展开**：
   经对待定系数的求解（或裂项），有：
   $$\frac{1}{(1-u)(1+u)^2} = \frac{1/4}{1-u} + \frac{1/4}{1+u} + \frac{1/2}{(1+u)^2}$$
   代入积分：
   $$
   \begin{aligned}
   \text{原式} &= -\frac{1}{2} \int \left( \frac{1/4}{1-u} + \frac{1/4}{1+u} + \frac{1/2}{(1+u)^2} \right)\,du \\
   &= \frac{1}{8}\ln|1-u| - \frac{1}{8}\ln|1+u| + \frac{1}{4(1+u)} + C \\
   &= \frac{1}{8}\ln\frac{1-\cos x}{1+\cos x} + \frac{1}{4(1+\cos x)} + C
   \end{aligned}
   $$

#### 【例 15】计算 $\int \frac{dx}{\cos x(1+\sin x)}$
**解析**：
被积函数关于 $\cos x$ 呈奇对称。
1. **分子分母同乘 $\cos x$**：
   $$\int \frac{\cos x\,dx}{\cos^2 x(1+\sin x)} = \int \frac{\cos x\,dx}{(1-\sin^2 x)(1+\sin x)} = \int \frac{\cos x\,dx}{(1-\sin x)(1+\sin x)^2}$$
2. **换元法**：令 $u = \sin x$，则 $du = \cos x\,dx$。
   $$\text{原式} = \int \frac{du}{(1-u)(1+u)^2}$$
3. **部分分式展开与求积分**（利用例14中相同的裂项系数）：
   $$
   \begin{aligned}
   \text{原式} &= \int \left( \frac{1/4}{1-u} + \frac{1/4}{1+u} + \frac{1/2}{(1+u)^2} \right)\,du \\
   &= -\frac{1}{4}\ln|1-u| + \frac{1}{4}\ln|1+u| - \frac{1}{2(1+u)} + C \\
   &= \frac{1}{4}\ln\left|\frac{1+\sin x}{1-\sin x}\right| - \frac{1}{2(1+\sin x)} + C
   \end{aligned}
   $$

#### 【例 16】计算 $\int \frac{dx}{\sin x(\sin x + \cos x)}$
**解析**：
被积函数中同时改变 $\sin x, \cos x$ 的符号，表达式不变，属于偶对称对称型。
1. **分子分母除以 $\sin^2 x$**，转化为 $\cot x$ 和 $\csc^2 x$ 的结构：
   $$\int \frac{dx}{\sin x(\sin x+\cos x)} = \int \frac{\frac{1}{\sin^2 x}}{1 + \frac{\cos x}{\sin x}}\,dx = \int \frac{\csc^2 x}{1 + \cot x}\,dx$$
2. **凑微分**：由于 $d(\cot x) = -\csc^2 x\,dx$，故：
   $$\text{原式} = -\int \frac{d(1+\cot x)}{1+\cot x} = -\ln|1+\cot x| + C$$
3. **化简结果**：
   $$-\ln|1+\cot x| + C = -\ln\left| \frac{\sin x + \cos x}{\sin x} \right| + C = \ln\left| \frac{\sin x}{\sin x + \cos x} \right| + C$$

## 3. 简单无理函数积分 $\int R\left(x, \sqrt[n]{\frac{ax+b}{cx+d}}\right)\,dx$

被积函数含有分数线性因式的根式。

### 核心方法：根式代换（无理代换）
为了消去根号，直接令整个根式为新变量：
$$t = \sqrt[n]{\frac{ax+b}{cx+d}}$$
由此可解出 $x$ 为 $t$ 的有理函数：
$$t^n = \frac{ax+b}{cx+d} \implies x = \psi(t)$$
同时可求出 $dx = \psi'(t)\,dt$，代入后将无理积分转化为**有理函数积分**。

### 特殊技巧
对于某些特定形式，可尝试**分子有理化**，配合拆分凑微分，可以极大简化计算。

---

### 简单无理函数典型例题

#### 【例 17】计算 $\int \frac{1}{x} \sqrt{\frac{x+1}{x}}\,dx$
**解析**：
* **方法一（根式代换）**：
  令整个根式为 $t$：
  $$t = \sqrt{\frac{x+1}{x}} = \sqrt{1 + \frac{1}{x}}$$
  两边平方得：
  $$t^2 = 1 + \frac{1}{x} \implies x = \frac{1}{t^2-1}$$
  求导得微分：
  $$dx = d\left(\frac{1}{t^2-1}\right) = -\frac{2t}{(t^2-1)^2}\,dt$$
  代入积分式：
  $$
  \begin{aligned}
  \int (t^2-1) \cdot t \cdot \left[ -\frac{2t}{(t^2-1)^2} \right]\,dt &= -2 \int \frac{t^2}{t^2-1}\,dt \\
  &= -2 \int \left( 1 + \frac{1}{t^2-1} \right)\,dt \\
  &= -2t - \ln\left| \frac{t-1}{t+1} \right| + C \\
  &= -2\sqrt{1+\frac{1}{x}} - \ln\left| \frac{\sqrt{1+\frac{1}{x}} - 1}{\sqrt{1+\frac{1}{x}} + 1} \right| + C
  \end{aligned}
  $$
  > **注**：利用共轭化简对数项：
  > $$\frac{\sqrt{1+1/x}-1}{\sqrt{1+1/x}+1} = \frac{\sqrt{x+1}-\sqrt{x}}{\sqrt{x+1}+\sqrt{x}} = (\sqrt{x+1}-\sqrt{x})^2 = 2x+1-2\sqrt{x^2+x}$$
  > 代回原式得到：
  > $$-2\sqrt{1+\frac{1}{x}} - \ln\left| 2x+1 - 2\sqrt{x^2+x} \right| + C$$

* **方法二（分子有理化与拆分）**：
  将分子根号化去：
  $$\frac{1}{x}\sqrt{\frac{x+1}{x}} = \frac{x+1}{x^2\sqrt{1+\frac{1}{x}}} = \frac{1}{x\sqrt{1+\frac{1}{x}}} + \frac{1}{x^2\sqrt{1+\frac{1}{x}}}$$
  对于第一部分，由于 $\frac{1}{x\sqrt{1+\frac{1}{x}}} = \frac{1}{x\sqrt{\frac{x+1}{x}}} = \frac{1}{\sqrt{x^2+x}}$；
  对于第二部分，由于 $d(1+\frac{1}{x}) = -\frac{1}{x^2}\,dx$；
  因此，拆分积分并求导：
  $$
  \begin{aligned}
  \int \frac{1}{x} \sqrt{\frac{x+1}{x}}\,dx &= \int \frac{dx}{\sqrt{x^2+x}} + \int \frac{dx}{x^2\sqrt{1+\frac{1}{x}}} \\
  &= \int \frac{d\left(x+\frac{1}{2}\right)}{\sqrt{\left(x+\frac{1}{2}\right)^2 - \frac{1}{4}}} - \int \frac{d\left(1+\frac{1}{x}\right)}{\sqrt{1+\frac{1}{x}}} \\
  &= \ln\left| x + \frac{1}{2} + \sqrt{x^2+x} \right| - 2\sqrt{1+\frac{1}{x}} + C
  \end{aligned}
  $$
  > **注**：方法一与方法二的结果完全等价，因为：
  > $$\ln\left| x + \frac{1}{2} + \sqrt{x^2+x} \right| = \ln\left| \frac{2x+1+2\sqrt{x^2+x}}{2} \right| = \ln(2x+1+2\sqrt{x^2+x}) - \ln 2$$
  > 而对数倒数有：
  > $$-\ln|2x+1-2\sqrt{x^2+x}| = \ln\left|\frac{1}{2x+1-2\sqrt{x^2+x}}\right| = \ln|2x+1+2\sqrt{x^2+x}|$$
  > 两者仅差一个常数项 $\ln 2$，验证了这两种方法的数学一致性。方法二巧妙避开了复杂的代数换元，展现了极高的解题美感。

## 4. 复习总结与做题方法论

1. **先判断结构**：
   - 分母可因式分解的代数式 $\implies$ **有理函数积分**。
   - 由 $\sin x$ 和 $\cos x$ 构成的比值式 $\implies$ **三角有理式积分**。
   - 含有一次分式根式 $\implies$ **简单无理函数积分**。
2. **特殊方法优先**：
   - 三角积分优先考虑**奇对称/偶对称换元**，最后才考虑万能代换。
   - 无理积分优先观察能否**分子有理化**拆分成已知凑微分模型，以缩短计算步骤。
3. **计算细心**：
   - 有理函数裂项时务必使用特殊值代入法快速核对系数。
   - 根式换元后，微分 $dx$ 的求解必须代入 $dt$，回代时要注意符号。
