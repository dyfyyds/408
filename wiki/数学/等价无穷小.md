# 等价无穷小

> 当 $x \to 0$ 时，两个无穷小量 $\alpha(x)$ 和 $\beta(x)$ 满足 $\lim\limits_{x \to 0} \frac{\alpha(x)}{\beta(x)} = 1$，则称 $\alpha \sim \beta$。

---

## 常见等价无穷小（$x \to 0$）

| 编号  | 等价关系                                                                                 |
| :-: | ------------------------------------------------------------------------------------ |
|  ①  | $\sin x \sim \tan x \sim \arctan x \sim \arcsin x \sim e^x - 1 \sim \ln(1+x) \sim x$ |
|  ②  | $(1+x)^\alpha - 1 \sim \alpha x$                                                     |
|  ③  | $1 - \cos x \sim \frac{1}{2}x^2$，推广：$1 - \cos^2 x \sim \frac{1}{2}x^2$               |
|  ④  | $a^x - 1 \sim x\ln a$                                                                |
|  ⑤  | $\tan x - \sin x \sim \frac{1}{2}x^3$                                                |
|  ⑥  | $x - \sin x \sim -\frac{1}{6}x^3$                                                    |
|  ⑦  | $\sin x - \arctan x \sim \frac{1}{6}x^3$                                             |
|  ⑧  | $x - \tan x \sim -\frac{1}{3}x^3$                                                    |
|  ⑨  | $x - \arctan x \sim \frac{1}{3}x^3$                                                  |
|  ⑩  | $\sec x - 1 \sim \frac{1}{2}x^2$                                                     |

---

## 等价无穷小代换原则

> 利用等价无穷小代换求极限。

### (1) 乘除关系：可以直接代换

若 $\alpha \sim \alpha_1, \beta \sim \beta_1$，则：

$$\lim\frac{\alpha}{\beta} = \lim\frac{\alpha_1}{\beta_1} = \lim\frac{\alpha}{\beta_1} = \lim\frac{\alpha_1}{\beta}$$

即乘除运算中，分子分母各自替换为等价无穷小，极限值不变。

### (2) 加减关系：分情况讨论

若 $\alpha \sim \alpha_1, \beta \sim \beta_1$，则：

**加法：** 若 $\lim\frac{\alpha_1}{\beta_1} \neq -1$，则 $\alpha + \beta \sim \alpha_1 + \beta_1$

**减法：** 若 $\lim\frac{\alpha_1}{\beta_1} \neq 1$，则 $\alpha - \beta \sim \alpha_1 - \beta_1$

---

## 使用注意

- **乘除运算**：任意等价无穷小均可直接替换，无需额外条件。
- **加法运算**：需验证 $\lim\frac{\alpha_1}{\beta_1} \neq -1$，否则 $\alpha + \beta$ 与 $\alpha_1 + \beta_1$ 不等价（如 $\sin x + \tan x \sim x + x = 2x$ 可以替换）。
- **减法运算**：需验证 $\lim\frac{\alpha_1}{\beta_1} \neq 1$，否则会产生误差（如 $\sin x - \tan x$ 中 $\sin x \sim x, \tan x \sim x$，但 $\sin x - \tan x \sim -\frac{1}{6}x^3 \neq x - x = 0$）。
- 等价无穷小替换仅适用于 $x \to 0$ 的极限过程。
- 复合函数替换：$\alpha(f(x)) \sim \beta(f(x))$ 当 $f(x) \to 0$ 时成立。
