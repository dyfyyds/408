# Cache映射计算

## 知识点

Cache 地址划分通常由主存地址、块大小、Cache 行数和组相联路数决定。

## 出错场景

- 块内地址位数算错。
- 组号位数与 Cache 行号位数混淆。
- 直接映射、全相联、组相联标签位不同。

## 识别信号

题干出现“主存容量、Cache 容量、块大小、几路组相联、地址格式”。

## 解题流程

1. 由块大小确定块内偏移位。
2. 由 Cache 容量/块大小得到 Cache 行数。
3. 由行数/路数得到组数。
4. 地址划分：标记位 + 组号/行号 + 块内偏移。

## 可复盘结论

先画地址字段，再代数计算；不要直接套记忆公式。
